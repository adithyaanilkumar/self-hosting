# self-hosting

## Caddy - Webserver
Install Caddy

```
sudo apt-get install caddy
```

Create a folder 

```
mkdir selfhost
```

```
cd selfhost
```

Create a html file

```
touch hello.html
```
```
nano hello.html
```
```
<h1> Hello from the other side ? </h1>
```
Ctrl + o and Ctrl + x

```
caddy
```
